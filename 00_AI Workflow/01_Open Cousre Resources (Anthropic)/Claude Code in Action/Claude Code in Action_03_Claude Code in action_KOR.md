#### 요약

Claude Code는 파일 읽기, 코드 작성, 명령 실행, 디렉토리 관리와 같은 일반적인 개발 작업을 처리하는 포괄적인 내장 도구 세트와 함께 제공됩니다. 하지만 Claude Code를 진정으로 강력하게 만드는 것은 이러한 도구를 결합하여 복잡한 다단계 문제를 어떻게 지능적으로 처리하는지입니다.
