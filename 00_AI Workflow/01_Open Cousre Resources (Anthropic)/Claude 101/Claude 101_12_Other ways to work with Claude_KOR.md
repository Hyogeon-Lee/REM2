이 과정의 시작에서 언급했듯이 Claude는 지능입니다. [Claude.ai](https://claude.ai/)는 그것으로 작업하는 한 가지 방법일 뿐입니다.

Claude는 또한 특정 워크플로우를 충족하기 위해 설계된 여러 특화된 도구에서 사용할 수 있습니다. 이 단원은 Claude와 함께 작업하는 네 가지 추가 방법을 소개하며, 각각은 특정 업무에 맞게 조정됩니다.

### Claude Code

![](https://www.youtube.com/watch?v=s-avRazvmLg)

Claude Code는 터미널, IDE, 브라우저 또는 Slack에서 작업하는 에이전트 코딩 도구입니다. 코드베이스를 이해하고, 명령을 실행하고, 자연 언어를 통해 전체 개발 워크플로우를 처리합니다.

**Claude Code를 사용할 때:**

- 일반 영어로 필요한 것을 설명하고 Claude가 코드를 작성하고, 테스트를 실행하고, 커밋을 만들기를 원할 때
- 오류 메시지를 붙여넣고 Claude가 코드베이스를 분석하여 문제를 파악하고 해결하려고 할 때
- 낯선 코드베이스를 탐색 중이고 다양한 부분이 어떻게 함께 작동하는지 물어보고 싶을 때
- lint 오류 수정, 병합 충돌 해결, 릴리스 노트 작성과 같은 지루한 작업을 자동화하려고 할 때
- 별도의 인터페이스로 전환하는 대신 기존 IDE 및 개발 도구와 함께 터미널에서 작업하기를 선호할 때

### Claude in Slack

![](https://www.youtube.com/watch?v=tI1uzSJuSxc)

Claude는 Slack에 직접 통합되어 있으므로 채널과 스레드에서 도움을 받거나 Slack 컨텍스트를 Claude 대화에 가져올 수 있습니다.

**Claude in Slack을 사용할 때:**

- Slack을 떠나지 않고도 메시지에 대한 응답 초안을 작성하거나, 긴 스레드를 요약하거나, 복잡한 논의를 분해하려고 할 때
- 관련 대화와 공유 문서를 Claude에 수집하여 회의를 준비하려고 할 때
- 새로운 팀에 온보딩 중이고 채널 기록을 검토하여 진행 중인 프로젝트를 이해하려고 할 때
- 버그 보고서 또는 기능 토론에서 코딩 작업을 직접 제공하려고 합니다. @Claude를 태그하면 주변 컨텍스트를 사용하여 Claude Code 세션을 시작할 수 있습니다
- 대화 중 업계 동향, 기술 개념, 회사 정보에 대한 빠른 답변이 필요할 때

### Claude for Excel

![](https://www.youtube.com/watch?v=8ZRTSIRWLu4)

Claude for Excel은 Microsoft Excel의 사이드바를 통해 Claude를 직접 가져와서 대화를 통해 스프레드시트를 분석, 이해, 수정할 수 있게 합니다.

**Claude for Excel을 사용할 때:**

- 복잡한 다중 탭 통합 문서로 작업 중이고 특정 공식이나 계산 흐름이 시트 전체에 어떻게 작동하는지 이해하려고 할 때
- 모델의 공식 종속성과 관계를 보존하면서 가정 또는 입력을 업데이트해야 할 때
- #REF!, #VALUE!, 순환 참조와 같은 스프레드시트 오류를 디버깅하고 Claude가 오류를 소스까지 추적하고 수정안을 제안할 때
- 적절한 공식 구조를 유지하면서 데이터로 새 스프레드시트를 생성하거나 기존 템플릿을 채우려고 할 때
- 데이터를 시각화하기 위해 피벗 테이블이나 차트를 신속하게 구축하려고 할 때

### Claude for Chrome

![](https://www.youtube.com/watch?v=IypXvHej9eY)

Claude for Chrome은 Google Chrome에 Claude를 사이드바로 추가하는 브라우저 확장입니다. 당신이 작업 중인 것을 관찰하고 브라우저 내에서 직접 조치를 취할 수 있습니다.

**Claude for Chrome을 사용할 때:**

- 찾아보면서 기사, 연구 논문, 웹 페이지를 요약하려고 할 때
- 이메일 응답 초안을 작성하거나 받은 편지함을 관리하는 데 도움이 필요할 때
- 반복적인 양식을 작성하고 프로세스를 자동화하려고 할 때
- 웹 사이트 기능을 테스트하거나 각 단계를 수동으로 클릭하지 않고 다중 단계 워크플로우를 탐색하려고 할 때
- 탭과 작업 사이를 이동하면서 컨텍스트를 유지하는 검색 어시스턴트가 필요할 때. 이는 틈새 내부 도구, CRM, 대시보드에서 컨텍스트를 끌어올 때 좋습니다.

**중요 참고:** Claude for Chrome은 현재 연구 미리보기 중입니다. Anthropic은 신뢰할 수 있는 웹 사이트에서 낮은 위험 작업에 사용하는 것을 권장합니다. 확장은 구매나 개인 데이터 공유와 같은 높은 위험 조치를 취하기 전에 권한을 요청하며, 특정 웹 사이트 범주 (금융 서비스, 성인 콘텐츠)는 기본적으로 차단됩니다.

### 요약

이러한 각 도구는 Claude의 기능을 작업하는 특정 환경으로 확장합니다:

| 도구 | 최고의 선택 | 실행 위치 |
| --- | --- | --- |
| Claude.ai | 일반 작업, 연구, 작성, 분석, 파일 생성 | 웹, 데스크톱, 모바일 앱 |
| Claude Code | 소프트웨어 개발, 코드베이스 탐색, git 워크플로우 | 터미널/명령줄, IDE, 브라우저 |
| Claude Cowork | 복잡한 다중 단계 작업: 연구 브리프, 문서 생성, 파일 조직, 데이터 분석 | 데스크톱 (및 Dispatch를 통한 모바일 앱) |
| Claude/Claude Code in Slack | 팀 협력, 회의 준비, 컨텍스트 내 빠른 답변 | Slack 워크스페이스 |
| Claude for Excel | 스프레드시트 분석, 재무 모델링, 공식 디버깅 | Microsoft Excel 사이드바 |
| Claude for Powerpoint | 슬라이드 생성, 프레젠테이션 편집, 서식 및 설계 | Microsoft Powerpoint 사이드바 |
| Claude for Chrome | 웹 연구, 이메일 관리, 브라우저 자동화 | Chrome 브라우저 사이드바 |
